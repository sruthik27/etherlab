# Version Mismatch

Mixes legacy PVST and Rapid PVST, then flaps the mixed-version trunk to capture both 802.1d and 802.1w BPDU behavior.

Artifacts land in `local/artifacts/stp/version-mismatch/`.

## Commands

- `bin/stp info version-mismatch`
- `bin/stp run version-mismatch`
- `bin/stp status version-mismatch`
- `bin/stp down version-mismatch`
