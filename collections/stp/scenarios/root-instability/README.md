# Root Instability

Pauses and unpauses the root bridge in a triangle to record transient root changes.

Artifacts land in `local/artifacts/stp/root-instability/`.

## Commands

- `bin/stp info root-instability`
- `bin/stp run root-instability`
- `bin/stp status root-instability`
- `bin/stp down root-instability`
