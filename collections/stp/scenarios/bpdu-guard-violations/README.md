# BPDU Guard Violations

Connects a rogue switch to a PortFast plus BPDU Guard access port and captures the offending BPDU before the violation silences the port.

Artifacts land in `local/artifacts/stp/bpdu-guard-violations/`.

## Commands

- `bin/stp info bpdu-guard-violations`
- `bin/stp run bpdu-guard-violations`
- `bin/stp status bpdu-guard-violations`
- `bin/stp down bpdu-guard-violations`
