# Root Misconfiguration

Injects superior PVST+ BPDUs with a non-standard raw priority so the active root wins by default rather than intentional priority design.

Artifacts land in `local/artifacts/stp/root-misconfiguration/`.

## Commands

- `bin/stp info root-misconfiguration`
- `bin/stp run root-misconfiguration`
- `bin/stp status root-misconfiguration`
- `bin/stp down root-misconfiguration`
