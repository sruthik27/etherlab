# Path Cost Conflicts

Captures a dual-homed downstream switch on VLAN 10 and 20 to expose bridge-ID tie-breaking and extreme path-cost outliers.

Artifacts land in `local/artifacts/stp/path-cost-conflicts/`.

## Commands

- `bin/stp info path-cost-conflicts`
- `bin/stp run path-cost-conflicts`
- `bin/stp status path-cost-conflicts`
- `bin/stp down path-cost-conflicts`
