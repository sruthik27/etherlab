# Dispute Detection

Injects inferior designated BPDUs onto an otherwise forwarding Rapid PVST segment to capture dispute-style disagreement.

Artifacts land in `local/artifacts/stp/dispute-detection/`.

## Commands

- `bin/stp info dispute-detection`
- `bin/stp run dispute-detection`
- `bin/stp status dispute-detection`
- `bin/stp down dispute-detection`
