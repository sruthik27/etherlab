# BPDU Silence

Pauses the root bridge for longer than max-age to create a BPDU gap on the downstream root port.

Artifacts land in `local/artifacts/stp/bpdu-silence/`.

## Commands

- `bin/stp info bpdu-silence`
- `bin/stp run bpdu-silence`
- `bin/stp status bpdu-silence`
- `bin/stp down bpdu-silence`
