# Proposal Agreement Failure

Forces an RSTP re-proposal while dropping return agreements, then injects an orphan agreement after the proposal window.

Artifacts land in `local/artifacts/stp/proposal-agreement-failure/`.

## Commands

- `bin/stp info proposal-agreement-failure`
- `bin/stp run proposal-agreement-failure`
- `bin/stp status proposal-agreement-failure`
- `bin/stp down proposal-agreement-failure`
