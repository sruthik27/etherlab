# Timer Misconfigurations

Injects a superior root with inconsistent hello, max-age, and forward-delay timers and validates them against IEEE timer relationships.

Artifacts land in `local/artifacts/stp/timer-misconfigurations/`.

## Commands

- `bin/stp info timer-misconfigurations`
- `bin/stp run timer-misconfigurations`
- `bin/stp status timer-misconfigurations`
- `bin/stp down timer-misconfigurations`
