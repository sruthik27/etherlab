# Slow Reconvergence

Suppresses BPDUs on the primary path while keeping the link up so failover is timer-driven.

Artifacts land in `local/artifacts/stp/slow-reconvergence/`.

## Commands

- `bin/stp info slow-reconvergence`
- `bin/stp run slow-reconvergence`
- `bin/stp status slow-reconvergence`
- `bin/stp down slow-reconvergence`
