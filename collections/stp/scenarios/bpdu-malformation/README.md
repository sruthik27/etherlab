# BPDU Malformation

Injects unknown-version, invalid-type, out-of-range-timer, and truncated BPDUs and decodes them with raw hex output.

Artifacts land in `local/artifacts/stp/bpdu-malformation/`.

## Commands

- `bin/stp info bpdu-malformation`
- `bin/stp run bpdu-malformation`
- `bin/stp status bpdu-malformation`
- `bin/stp down bpdu-malformation`
