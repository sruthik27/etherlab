# BPDU Rate Anomalies

Injects a bursty, jittery synthetic bridge stream and measures inter-arrival variance and rate drift against the hello timer.

Artifacts land in `local/artifacts/stp/bpdu-rate-anomalies/`.

## Commands

- `bin/stp info bpdu-rate-anomalies`
- `bin/stp run bpdu-rate-anomalies`
- `bin/stp status bpdu-rate-anomalies`
- `bin/stp down bpdu-rate-anomalies`
