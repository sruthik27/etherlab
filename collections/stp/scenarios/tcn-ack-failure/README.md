# TCN ACK Failure

Uses a legacy PVST downstream access-link flap and flags TCNs that are not acknowledged within the expected timeout window.

Artifacts land in `local/artifacts/stp/tcn-ack-failure/`.

## Commands

- `bin/stp info tcn-ack-failure`
- `bin/stp run tcn-ack-failure`
- `bin/stp status tcn-ack-failure`
- `bin/stp down tcn-ack-failure`
