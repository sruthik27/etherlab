# Collections

Collections are the public, browsable units of this repo.

Each collection should own its own:

- top-level README
- scenarios or labs
- collection-specific workflows
- collection-specific command wrapper when needed

## Available Now

- `lag`: EtherChannel baseline labs and mismatch or failure scenarios
- `stp`: spanning-tree baseline labs and anomaly/fault scenarios

## Planned Directions

- DHCP labs
- DNS labs
- switching and edge-security labs
- additional wired-networking experiments
